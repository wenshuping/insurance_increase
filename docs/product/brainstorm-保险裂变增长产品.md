## 二、传播障碍根因分析

> **保险最大的传播障碍不是产品复杂，而是「信任赤字 + 场景割裂 + 利益不透明」**

| 现状 | 用户感知 |
|------|---------|
| 业务员群发产品海报 | 广告轰炸，屏蔽 |
| 转发保险条款链接 | 我在替你打工 |
| 晒保单 | 炫耀 + 无关我 |
| 转介绍返现 | 利益驱动，不可信 |

**关键洞察：** 用户愿意传播的本质是——传播这件事让「我」显得聪明/有爱心/有价值

## 三、产品核心定位

```
产品名称（待定）：保险家 / 保盾 / 家盾

核心定位：
  不是卖保险的工具
  而是传播「风险认知」的媒体平台
  业务员是内容节点，客户是传播网络

核心价值主张：
  让每个业务员都成为家庭保障顾问
  让每个客户都有「帮朋友」的理由和工具
```

## 四、增长飞轮（完整闭环）

```
业务员发布「保障缺口测评」链接
         ↓
潜在客户B：30秒测出家庭保障缺口
         ↓
结果显示：「你的缺口比同龄人高X%」（焦虑锚点）
         ↓
B分享给家人/同事（完善家庭报告/提醒朋友/职场传播）
         ↓
每个新用户自动归属原始业务员
业务员实时收到「线索热度通知」
         ↓
成交 → 客户变新节点 → 理赔故事接力 → 二次传播
         ↓
（飞轮加速，K因子目标 > 0.5）
```

## 五、功能优先级规划

### MVP 阶段（验证传播意愿）

#### 功能1：家庭保障缺口测评 H5
- 30秒填写，生成家庭风险雷达图（红/黄/绿）
- 结果页锚点比较：「你的缺口比同龄人高X%」
- 邀请家人共同填写，解锁完整家庭报告
- 裂变逻辑：拉家人完善报告 = 自然行为，非广告

#### 功能2：傻瓜可视化计划书
- 录入客户信息，30秒生成漫画式图解计划书
- 可视化呈现：风险缺口 → 方案填补 → 月费 → 理赔预期
- 客户可转发，自己回家能讲清楚

### Growth 阶段（验证转化）

#### 功能3：理赔故事机
- 脱敏理赔卡片：「37岁，确诊肺癌，30天到账38万」
- 时间线+金额+客户感言（文字/语音）
- 底部引导：「测测你的保障」→ 进入裂变测评
- 传播动机：利他（帮朋友避坑）+ 社会证明

#### 功能4：保险翻译官
- AI将条款翻译成一句话人话版
- 生成「避坑图鉴」系列竖版长图
- 底部标签：「由 [业务员名字] 为你整理」
- 传播动机：用户转发知识 = 显示聪明

#### 功能5：保单礼物（赠险）
- 赠送微额意外险或健康测评包（真实承保，合规）
- 精美节日/生日卡片，适合朋友圈/微信
- 收礼者激活后自动进入业务员线索池
- 分享语：「XXX关心你，送你一份保障」

### 壁垒阶段（建立情感护城河）

#### 功能6：保险家书（最颠覆性概念）
> 父母给子女配置保障时留言，子女收到「一份来自爸妈的保障礼物」通知
> 保险从商品变情感载体，每份保单背后都有可传播的故事
> 竞品无法抄袭，核心壁垒是情感数据积累

#### 功能7：业务员个人品牌主页
- 展示：从业年限/服务客户数/理赔记录/客户评价
- 可分享到朋友圈（业务员主动传播，因为帮自己打广告）
- 同行看到 → 主动来问用什么工具 → 口碑传播

#### 功能8：新人每日任务标准化
- 复制团队长成功路径为每日任务清单
- 新人打开就知道今天该干什么（见客量/内容/学习）
- 团队长实时看漏斗数据：谁卡在哪个环节
- 解决核心痛点：90天脱落率高

## 七、客户三类画像与差异化触达

### 小王（28岁，互联网男）
- **痛点：** 觉得复杂、不信任、不敢面对风险
- **触达方式：** 量化工具「如果明天出事，你妈怎么办」
- **传播动机：** 内容让他显得聪明（反常识知识）
- **理想产品：** 算一道题，结果可怕，方案极简，无销售

### 林姐（42岁，教师）
- **痛点：** 买了但看不懂，理赔失望，心有余悸
- **触达方式：** 先帮她看懂已有保险，建立信任
- **传播动机：** 真实案例（朋友得病触发），工具类内容
- **理想产品：** 保险管家（整理已有+年度体检+理赔协助）

### 张总（50岁，企业主）
- **痛点：** 知道重要，不信任销售利益不一致
- **触达方式：** 带有价值信息来，以专家身份请教他
- **传播动机：** 小圈子私信分享，「让同行感到我更早想清楚」
- **理想产品：** 独立顾问（收服务费而非提成，利益一致）

## 八、5种裂变机制（增长黑客版）

| 机制 | 对标案例 | 裂变原理 |
|-----|---------|---------|
| 保障缺口测评 | 网易云年度报告 | 相对剥夺感 → 让家人也测 |
| 理赔故事接力 | 拼多多砍一刀 | 社交信用背书 → 需关系链激活 |
| 家庭保障共建 | Dropbox邀请送空间 | 功能完整性解锁 → 自然拉人 |
| 职场保障地图 | 脉脉职业身份 | 圈层传播 → 同行业人群 |
| 保单礼物 | 微信红包 | 礼物经济学 → 送礼者有面子 |

## 九、3个内容传播钩子

### 钩子1：「惊吓型」可视化报告
```
「假如你今天失去劳动能力，你的家庭还能撑多少个月？」

你的月支出：15,000元
你的现有保障：0元
紧急储备金：可撑 3 个月
之后：房贷/孩子教育/父母赡养 同时断供
```
传播动机：家庭崩塌恐惧最强，本能反应是让爱人/父母也看

### 钩子2：「知识型」反常识内容
- 「先给孩子买保险 = 财务逻辑错误（正确顺序是什么）」
- 「百万医疗险的免责条款，你一定没仔细看第7条」
传播动机：揭露错误比传授知识传播率高3倍

### 钩子3：「情感型」理赔故事
> "2024年3月，老王被确诊为早期肺癌。42天后，他收到了28万理赔款。
> 他说：'那天我第一次觉得，当初那个买保险的决定是对的。'"

传播动机：故事 + 信任验证，父母群/中年同事群高转发率

## 十、合规红线清单

| 禁止行为 | 合规替代方案 |
|---------|------------|
| 承诺收益/夸大保障 | 只展示保障缺口，不做收益承诺 |
| 现金返佣给消费者 | 服务权益（体检/咨询/法律顾问）积分兑换 |
| 虚假场景引流 | 内容标明「保险产品宣传」 |
| 恐吓/饥饿营销 | 数据呈现代替话术煽动 |
| 诱导退保搭单 | 系统禁止此类表述和流程 |
| 未授权收集家庭信息 | 明示授权+PIPL合规设计 |

## 十一、冷启动策略

### 目标：第一批100个业务员

**不找顶级业务员，找「委屈的中间层」**
- 特征：入行2-5年，50-200存量客户，对现有工具不满，月收入1-3万
- 在哪：保险从业者论坛、公司培训班、腰部业务员微信群

**三步走：**

| 阶段 | 目标人数 | 策略 |
|-----|---------|-----|
| 第一步 | 前30人 | 以「效率工具」切入（客户管理+报告生成），免费6个月换深度反馈 |
| 第二步 | 30-70人 | 邀请制+内部圈子稀缺感，建种子群（满100人关闭） |
| 第三步 | 70-100人破圈 | 找1个明星案例（多成交X单），在从业者社群定向投放 |
