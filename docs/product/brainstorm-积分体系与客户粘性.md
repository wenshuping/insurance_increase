## 一、核心共识

| 视角 | 最大洞察 |
|-----|---------|
| 产品经理+增长黑客 | 积分本质不是「奖励购买」，而是「奖励风险认知传播」，每个积分触发点=一次保险教育机会 |
| 刘哥（业务员） | 积分活动的核心价值：给了我一个「正当理由」联系存量客户，而不是骚扰 |
| 陈总监（团队长） | 客户活跃度排行榜才是真正驱动业务员执行的引擎，经费挂钩活跃度指标 |

| 合规法务 | 合规安全区间：服务类权益 + 小额实物 + 平台预算支出（非保险公司资金） |

## 二、积分体系全架构

### 积分获取路径

| 行为 | 积分 | 设计意图 |
|-----|-----|---------|
| 完成家庭保障缺口测评 | 500分 | 高价值首触点，数据沉淀 |
| 购买保险产品 | 保费×10分 | 绑定GMV |
| 成功转介绍1人（对方注册） | 800分 | 核心裂变动力 |
| 被转介绍人成交 | 额外500分（双向奖励） | K因子加速 |
| 每日保险知识打卡（答对） | 20分/天，连打7天×2倍 | 习惯养成 |
| 参加线下活动签到 | 300分 | 线下流量激活 |
| 分享测评结果到朋友圈（验证曝光） | 50分 | 传播素材 |

> **MVP阶段只做3个：** 学习打卡（30分/天上限）+ 活动签到（50分）+ 体检券兑换（200积分）

### 积分消耗路径（兑换层级）

| 层级 | 兑换品 | 积分 | 目标用户 |
|-----|-------|-----|---------|
| 大众层 | 咖啡券/视频会员/电商优惠券 | 100-300分 | 小王 |
| 中端层 | 体检套餐/法律咨询1次/药品折扣 | 600-1500分 | 林姐 |
| 高端层 | 仅限邀请的私享财富沙龙名额 | 2000-3000分 | 张总 |
| 保险增值 | 1年意外险/续费保费抵扣（上限20%） | 1200分 | 全客群 |

> **关键合规提示：** 保费抵扣操作须由持牌保险公司授权；实物商品单次≤200元；现金/购物卡/现金等价物禁止

### 等级体系

```
普通会员(0) → 银卡(1000) → 金卡(3000) → 钻石(8000) → 守护家(20000)
```

「守护家」享有：优先理赔通道 + 专属客服 + 年度体检
（命名本身即传播素材——用户会在朋友圈晒「我成为守护家了」）

### 积分过期机制

- 基础：12个月不活跃则过期
- 任意活跃行为（打卡/分享/参与活动）自动续期6个月
- 每年12月发送「你的XXX分将于30天后过期」推送 → 年末促活神器

## 三、保险知识学习系统

### 内容选题公式（用户才会主动学的内容）

> 「你身边发生过的 + 你以为有保但其实没有的 + 用数字说话的」

- ✅ 「王女士住院了，结果发现她买的产品实际赔付0元——为什么？」
- ✅ 「你家收入断3个月，房贷还得上吗？」
- ❌ 避免：「重疾险和医疗险的区别是……」（教科书感，直接关掉）

### 打卡机制设计

- 每日3题，答对才得分（错了可重做）
- **不做全国榜，做「业务员客户圈小排行榜」**（10人圈子竞争，对标微信步数）
- 连击提示（连打N天）→ 留存率参考数据+40%
- 答题后自然引导：「你家是否有同样的保障缺口？点击测评」→ 学习→测评→焦虑→咨询，链路一气呵成

### 学习→转化的自然流向

```
答题（获积分）→ 看到自己的保障缺口 → 触达业务员 → 成交 → 理赔故事 → 再次传播
```

## 四、活动体系设计

### 活动类型与门槛

| 活动类型 | 形式 | 积分 | 优先级 |
|---------|-----|-----|-------|
| 线上知识测评H5 | 30秒，生成报告，分享传播 | 200分 | MVP首选 |
| 线上直播问答 | 45分钟，实时答题抢积分 | 100分 | Phase 2 |
| 线下健康讲座 | 专业医生主讲，签到积分 | 300分 | Phase 2 |
| 线下财富沙龙 | 律师/理财师主讲，邀请制 | 500分 | Phase 3（张总专属） |
| 亲子活动 | 周末家庭场景，低门槛裂变 | 300分 | Phase 2 |

### 裂变绑定机制

不是简单「带人来翻倍」，而是**三段奖励分开推送**，制造持续惊喜感：

1. 我参加活动 → 300分（立刻到账）
2. 我邀请的人来了 → 再得300分（次日推送：「你的朋友参加了！」）
3. 对方购买产品 → 追加500分（24小时限时通知：「恭喜你获得成交奖励」）

## 五、家庭积分池（最创新机制）

**核心设计：** 家庭成员可共享一个积分池，合并积分兑换家庭级权益

**为什么会work：**
- 保险天然是家庭决策 → 积分也变成家庭话题
- 「老公把积分转给我，换个家庭体检」→ 业务员被动进入家庭对话
- 家庭池积分数字更大 → 舍不得过期心理更强
- 有了共同目标（兑换家庭体检套餐）→ 更强的留存动力

**对标案例：** 招商银行「家庭账户」功能，留存率比个人账户高31%

**K因子加速：** 有家庭积分池的用户，邀请家庭成员的概率是普通用户的4倍
（「帮家人攒积分」比「推销保险」更好的邀请话术）

## 六、三类客户差异化钩子

### 小王（28岁，互联网男）
- **他的心理驱动：** 信息优越感（比别人先知道）
- **触发点：** 真实理赔案例 + 人话版解读，5分钟看完
- **有效兑换品：** 咖啡券/健身年卡/游戏点卡（生活方式品）
- **让他直接拉黑的：** 早8点推送 + 话术像卖货 + 中老年养生礼包

### 林姐（42岁，教师）
- **她的心理驱动：** 被重视、被听见（不是被教育）
- **触发点：** 「你的保障盲区测评」——让她发现自己不知道的
- **她拉闺蜜来的条件：** 活动有「实用产出」（属于自己的保障清单），不是热闹讲座
- **积分对她的意义：** 我不是可以被随便忽悠的人

### 张总（50岁，企业主）
- **他的心理驱动：** 圈子筛选 + 引荐人身份
- **触发点：** 「参加的都是什么人？主讲人是谁？」（积分是入场门槛，不是目的）
- **他推荐给生意伙伴的时机：** 活动结束后他觉得「这个圈子值得带朋友进来」
- **必须给他的：** 「引荐人身份」而不是普通参与者

## 七、合规红线清单

| 行为 | 结论 | 合规替代方案 |
|-----|-----|-----------|
| 积分兑换现金/购物卡 | ❌ 禁止 | 服务类权益（体检/咨询） |
| 积分直接抵扣保费 | ❌ 禁止（中介无权操作） | 需保险公司授权配合 |
| 积分兑换实物商品 | ✅ 有条件允许 | 单次≤200元，平台自有预算，留存发票 |
| 积分兑换服务（体检/法律） | ✅ 最安全 | 以「平台会员权益」名义，合同签在平台主体 |
| 讲座主讲保险知识 | ✅ 须持证业务员主讲 | 禁止炒停/夸大收益/恐吓 |
| 收集健康问卷数据 | ⚠️ 敏感信息 | 需单独弹窗授权，禁止自动化决策推荐产品 |
| 业务员代替客户操作积分 | ❌ 禁止 | 实名答题+签到定位验证 |

## 八、运营指标体系

### MVP阶段关键指标（3个月验证）

| 假设 | 验证指标 | 健康值 |
|-----|---------|-------|
| 学习积分能提升留存 | 月活提升 | 基准+20% |
| 体检券兑换率验证权益吸引力 | 兑换率 | 15%~25% |
| 业务员组织活动意愿 | 人均活动次数 | 0.3→1次/月 |

### 长期运营健康指标

| 指标 | 健康值 |
|-----|-------|
| 月积分参与率 | ≥30%活跃用户 |
| 积分兑换率 | 15%~25% |
| 活动人均获客成本 | ≤80元/新客 |
| 积分平均兑换周期 | ≤45天 |

### 防刷分机制

- 同一设备24小时积分上限50分
- 答题类设置最低完成时长（60秒/题）
- 活动签到需定位验证
- 业务员账号与客户积分规则隔离

## 十、冷启动策略（最小可行积分体系）

### 第一个活动选型

**「保险知识测评」线上H5** ——原因：
- 零场地成本
- 天然传播（分享测评结果）
- 合规无风险
- 数据沉淀用户知识图谱

### 第一版积分体系（只做3件事）

1. 学习积分（完成1篇知识文章得10分，每日上限30分）
2. 活动签到积分（线下/线上活动签到50分）
3. 兑换层：只上线体检优惠券1档（200积分换价值128元体检券）
