# 👥 客户角色

> 模拟真实用户体验产品，用于测试、用研、体验验证。

## Skills
- `customer-persona` — 客户角色模拟（多年龄/性别/身份）
- `insurance-expert` — 保险产品知识
- `x-article-publisher` — 文章发布到 X

## 内置角色
| 群体 | 角色 |
|------|------|
| 青年 | 小王（互联网男）· 小李（应届女生）· 阿杰（自由职业）· 小美（已婚白领） |
| 中年 | 张总（企业主）· 王姐（全职主妇）· 老陈（国企中层）· 林姐（教师） |
| 老年 | 王阿姨（退休工人）· 李大爷（退休干部） |
| 特殊 | 外卖小哥阿明 · 海归Lucy · 宝妈小雪 |
